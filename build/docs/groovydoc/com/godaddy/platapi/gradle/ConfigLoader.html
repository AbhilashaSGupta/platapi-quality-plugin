Package: com.godaddy.platapi.gradle

[Groovy] Class ConfigLoader

    • com.godaddy.platapi.gradle.ConfigLoader

    • Loads configuration files either from custom configs directory (quality.configDir) or from classpath. When loading from classpath all files are copied (and cached) in build/quality-configs/.

      To avoid problems with clean task, default configs are copied only before actual task run. All config file getters contains optional parameter copyDefaultConfig to prevent actual file copying on configuration phase.
